Instructions

Here are a few instructions to make it easier to further develop this template.

01 — The fixed home hero

As you may have noticed, the hero section on the home page is fixed and gets covered by the rest of the page while scrolling down. In order for it to be visible (before scrolling down), it's positioned inside a "Fixed Hero Visibility Block". Both the hero section and the visibility block is 100 VH in height, which is the maximum height they're able to be. Any additional height won't be visible because of its fixed position.

If you want to have a hero section with less height, it's important to change the height on both the hero section and the visibility block. Otherwise you might run into unintended bugs.

02 — Button text

The button hover interaction used throughout this template makes it hard to edit the text of buttons correctly. In order to change the text of a button, you need to change both the initial text inside the button (with the class Button Text) as well as the secondary text element (with the class Button Text Absolute).

This is however accomplished by first giving the Button Text Absolute class a static position, change its text and finally give it back its absolute position.

03 — Navigations

There are two nav components used in this template. The first one is called "Nav", and has a button with a white background color. This one is meant to be used on sections with a non-white background color.

The second one is called "Nav On White Background", which, as its name suggests, is intended to be used on sections with a white background color.

04 — Smooth scrolling

This template uses the Lenis smooth scroll library to achieve the smooth scrolling effect. It works well together with all kinds of native Webflow scroll interactions.

If you don't like this effect, feel free to remove it from the "before body" custom code in the site settings. You're also able to customize it according to the official Lenis documentation.
